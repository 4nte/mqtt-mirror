## Highlights

- **Topic rewriting** — `--topic-prefix` and `--topic-replace` flags to transform topic names during mirroring
- **Persistent sessions** — `--clean-session` flag to control MQTT session persistence across reconnects
- **Subscribe QoS control** — `--subscribe-qos` flag to set QoS level for source broker subscriptions
- **Multi-arch Docker images** — `linux/amd64` and `linux/arm64` builds

## Features

- Add `--topic-prefix` flag to prepend a prefix to all mirrored topics
- Add `--topic-replace` flag for `old:new` string replacements on topic names (repeatable)
- Add `--clean-session` flag (default `true`); set to `false` for MQTT persistent sessions with durable subscriptions
- Add `--subscribe-qos` flag to control QoS level when subscribing to the source broker
- Add `--publish-timeout` flag to configure publish timeout duration
